# aqi-data-analysis-python
Air Quality Index analysis across major Indian cities using Python (EDA + Trend Analysis)
Air Quality Index (AQI) Analysis Across Major Indian Cities

_Project Overview_
This project analyzes Air Quality Index (AQI) data across major Indian cities using Python. The goal is to understand pollution trends, identify seasonal patterns, and compare air quality levels between cities through Exploratory Data Analysis (EDA).

_Objectives_
Combine multiple city-wise AQI CSV datasets into a unified dataset
Clean and preprocess raw data
Perform statistical and time-series analysis
Identify seasonal pollution trends
Compare city-wise average AQI levels
Extract meaningful insights from environmental data

_Tools & Technologies Used_
Python
Pandas – Data cleaning and manipulation
Matplotlib – Data visualization
Seaborn – Statistical visualization
Jupyter Notebook

Key Insights
Delhi recorded the highest average AQI (~208), indicating severe pollution levels
Seasonal patterns were observed, with higher AQI values during winter months
Many cities frequently exceeded the safe AQI limit of 100
Southern cities like Bangalore and Chennai showed comparatively better air quality
AQI values show significant variability across time and location

_Limitations_
Only AQI values were analyzed (no pollutant-level breakdown)
Weather, traffic, and industrial factors were not considered
Data completeness depends on source CSV files

_Future Scope_
Predict AQI using Machine Learning models
Analyze individual pollutants (PM2.5, PM10, NO₂, SO₂)
Integrate weather data for deeper analysis
Build an interactive dashboard using Streamlit or Power BI
Perform anomaly detection for extreme pollution events
